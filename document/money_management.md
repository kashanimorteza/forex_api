# Money Management
```
مدیریت مالی استراتژی را بر عهده دارد
مشخص می‌کند که چه مقدار ریسک کند
چه مقدار از موجودی کلی را برداشت بکند
همچنین یک سری محدودیت برای استراتژی در نظر می‌گیرد
```



<!--------------------------------------------------------------------------------- Items --->
<br><br>

## Items

<!-------------------------- Balance -->
### Balance
```
مقدار پولی که استراتژی حق دارد از موجودی کل در اختیار بگیرد
```

<!-------------------------- Risk -->
### Risk
```
مشخص نمودن درصد حجم معامله از موجودی استراتژی
```

<!-------------------------- Limit Trade -->
### Limit Trade
```
تعداد پوزیشن‌هایی که یک استراتژی می‌تواند باز کند
```

<!-------------------------- Limit Profit -->
## Limit Profit
```
بستن تمام پوزیشن‌های باز و استاپ کردن استراتژی در صورتی که سود به این عدد برسد
```

<!-------------------------- Limit Loss -->
### Limit Loss
```
بستن تمام پوزیشن‌های باز و استاپ کردن استراتژی در صورتی که ضرر به این عدد برسد
```

<!-------------------------- Limit Stop -->
### Limit Stop
```
فریز شدن و باز نکردن پوزیشن جدید وقتی سود تمام پوزیشن‌های بسته شده به این عدد برسد
```
